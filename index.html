Tags HTML 5

explicação

<Header>

<Header> é uma tag usada para dar suporte introdutório ou navegacional podendo haver elementos de cabeçalho ou outros como um logo, seções de cabeçalho, formulário de pesquisa, etc.

<Section>

Uma <section> deve ser utilizada quando não possuir elementos específicos como uma lista de resultados de pesquisa ou a exibição de um mapa e seus controles não possuem elementos especificos podem ser colocados dentro de uma section.

nota:

<Section> é utilizada para documentos e a tag <div> para fins de estilo

<Article>

<Article> é uma composição independente em um documento, página, aplicação ou site. oque for destinado a ser usado de forma independente e reutizável, como por exemplo, um artigo de revista ou jornal, um post de um blog, um comentário enviado por um usuário, um gadget ou widget interativos, ou qualquer outra forma de conteúdo independente.

<Nav>

A tag <nav> é uma seção de uma página que irá servir como uma seção de links de navegação apontando para outras páginas ou outras áreas da página.

<Aside>

O elemento HTML <aside> representa uma seção de uma página que consiste de conteúdo que é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo. Essas seções são, muitas vezes, representadas como barras laterais.

<Main>

O elemento <main> define o conteúdo principal dentro do <body> em seu documento ou aplicação. Entende-se como conteúdo principal aquele relacionado diretamente com o tópico central da página ou com a funcionalidade central da aplicação. O mesmo deverá ser único na página, ou seja, dentro do elemento <main> não deverão ser incluidas seções da página que sejam comuns a todo o site ou aplicação

nota:

Só pode haver uma tag <main>.

<Figure>

O elemento HTML <figure> representa o conteúdo independente, frequentemente com uma legenda (<figcaption>), e é normalmente referido como uma única unidade. Enquanto ela está relacionada com o fluxo principal, sua posição é independente do fluxo principal.Normalmente, isso é uma imagem, uma ilustração, um diagrama, um trecho de código ou uma esquema que é referenciado no texto principal

nota:

Auxilia as tags <img> e <figcaption>.

<Footer>

O elemento HTML de Rodapé Normalmente um rodapé <footer>contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

<a>

O elemento <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada precisará indicar o destino do link.

nota:

E essencial no uso de páginas web.

<em>

O elemento HTML <em> marca o texto que tem ênfase.

<Strong>

O elemento HTML <strong> dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

nota:

também usado para destacar texto.

<Cite>

O elemento HTML <cite> representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

<Figcaption>

O elemento HTML Figcaption <figcaption> representa uma legenda ou uma legenda associada com uma figura ou ilustração descrita pelo resto dos dados do elemento <figure> que seu elemento pai.

nota:

<Figcaption> deve ser usado com o <Figure>.
